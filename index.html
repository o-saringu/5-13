プログラム復習コンテンツ

map()関数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(240, 120);　　

　　 　strokeWeight(12);

　　 }

　　

　　 void draw(){

　　 　background(204);

　　 　　if(keyCode==LEFT){

　　 　　stroke(102);

　　 　　line(mouseX, 0, mouseX, height);

　　 　　 stroke(0);

　　 　

　　 　　float mx=map(mouseX, 0, width, 60, 180);

　　 　　line(mx, 0, mx, height);

　　 }

説明文

　map()関数は、最小値と最大値を変換させる役割がある。

　map(a, b, c, d, e)

　　aを、範囲b~cから別の範囲d~eへと変換させる。

　　b~cの範囲をd~eの範囲に適応させる。

　例で説明をすると、例では2本の線がある。　

　1本目の線はカーソルに合わせて移動する。

　2本目の線は、map関数を用いて最小値、最大値を変換しているため、カーソルがx座標0の時に、x座標60と変換される。

　また、カーソルが一番右端の時は、変換されているため180座標になる。
